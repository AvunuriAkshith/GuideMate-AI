# 🌍 GuideMate AI

AI-powered tourist guide booking platform built using MERN + FastAPI + Socket.IO.

---

## 🚀 Features

### 👨‍💼 Guide Features
- Guide Dashboard
- Booking Management
- Tourist Chat
- Real-time Notifications
- Profile Management

### ✈️ Tourist Features
- Explore Travel Guides
- AI Travel Assistant
- AI Trip Planner
- Real-time Chat
- Saved Trips
- Reviews & Ratings
- Interactive Travel Map

### 🤖 AI Features
- Gemini AI Chatbot
- AI Itinerary Planner
- Smart Travel Suggestions

### 💬 Real-time Features
- Socket.IO Chat
- Guide ↔ Tourist Messaging
- Live Chat Rooms

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Socket.IO Client
- React Router
- Lucide React

### Backend
- FastAPI
- Python
- Socket.IO
- Motor (MongoDB)

### Database
- MongoDB Atlas

### AI
- Google Gemini API

## ⚙️ Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
